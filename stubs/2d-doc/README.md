# 2d-doc

This is an alias package for [py2ddoc](https://pypi.org/project/py2ddoc/).

Installing `2d-doc` automatically installs `py2ddoc`.

```bash
pip install 2d-doc  # installs py2ddoc
pip install py2ddoc # canonical name
```

```python
from py2ddoc import TwoDDoc
```

Please use `py2ddoc` directly for the canonical package.
